Links

1. Hyperlink

The HTML <a> tag defines a hyperlink. Clicking on a hyperlink allows you to go to the link's destination.

Absolute URL Clicking on HOME will lead you to the HOME page.
It can be used as follows:

<a href="/index.html">HOME</a>

Relative URL Clicking on Visit Google will lead you to Google's homepage.
It can be used as follows:

<a href="https://google.com/">Visit Google</a>

Target Attribute
_self - Default. Opens the document in the same window/tab as it was clicked.
_blank - Opens the document in a new window or tab.
_parent - Opens the document in the parent frame.
_top - Opens the document in the full body of the window.
Note! By default, the linked page will be displayed in the current browser window.
It can be used as follows:

<a href="https://google.com/" target="_blank">Visit Google</a>

Image as a Link Clicking on will lead you to Google's homepage.
It can be used as follows:

<a href="https://google.com/">
  <img src="smiley.gif" alt="Google Image" style="width:42px;height:42px;">
</a>

Link to an Email Address Clicking on Send email to someone@example.com create a link that opens the user's email program (to let them send a new email).
It can be used as follows:

<a href="https://google.com/">Send email to someone@example.com</a>

Link Titles Hovering on Visit Google will show you the information of the "Visit Google" link.
It can be used as follows:

<a href="https://google.com/" title="Go to Google">Visit Google</a>

Button URL Clicking on
It can be used as follows:

<a href="https://google.com/">Visit Google</a>

2. External Link

The HTML <link> tag defines the relationship between a document and an external resource.
It can be used as follows:

<link rel="stylesheet" href="/CSS/mainindex.css">

3. Navigation Links

The HTML <nav> tag is usyually used for the navigation bar.
It ususlly directs you to go to another page of the web.
Example is used in this page navigation bar.
It can be used as follows:

<nav>
 <a href="/html/">HTML</a>
 <a href="/css/">CSS</a>
 <a href="/js/">JavaScript</a>
</nav>

3. Link Bookmarks

Link Bookmarks ususlly directs you to go to a destination of a page.
Clicking on Go back to Links will get you to Links destiantion
It can be used as follows:
<a href="#Links">Go back to Links</a>
You can also add a link to a bookmark on another page. Clicking on Go to CSS will get you to Links destiantion.
It can be used as follows:
<a href="../CSS/index.html#css">Go to CSS</a>

Lists

An unordered list starts with the <ul> tag. Each list item starts with the <li> tag.
The list items will be marked with bullets (small black circles) by default:

It can be used as follows:
<ul>
 <li>Coffee</li>
 <li>Tea</li>
 <li>Milk</li>
</ul>

Output:
  • UnorderedList1
  • UnorderedList2
  • UnorderedList3
